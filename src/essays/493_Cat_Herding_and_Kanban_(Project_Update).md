As I have flashbacks to the Project Management (PM) life of cat herding, trying to get people to use a Kanban board so that the team knows what they’re working on, it is time for another post about the project.

There are still opportunities to join us and contribute to the project, either coding portions or adding other value to any of the 4 primary segments for phase 1:

1\. ICOM-based Seeds (The graph data structure from which systems iteratively grow):

Contributors (including non-technical) can add metadata to seed material via an associative exercise.

2\. Global Workspace (The “Subjective Experience” / motivational processing portion):

Anyone with a sufficiently deep technical background can help build this component to satisfy Global Workspace Theory, constrained only by the graph data and motivational system baked into it.

3\. Graded Loop for checking the outputs of any LLM-based tool or “agent” (A component to grant a basic degree of robust error-correction and “hallucination” mitigation that iteratively improves):

Anyone with a more typical level of computer science and coding background could contribute to building this functionality.

4\. Environment, Integration, and Deployment + Graph Data Conversion (Everything else, the glue and data conversions that hold it all together):

Again, anyone with a more typical level of computer science and coding background could contribute to building this functionality.

For now, I’m placed in the difficult situation of having to continue recruiting for the project until Phase 1 is complete. I’m meeting with potential interested parties to onboard on a rolling basis, so please reach out if you’re interested.

Given the laughable capacities that major companies still brag about today, I fully expect anyone who meaningfully contributes to the project to receive job offers from several companies. I won’t be entertaining any such offers, as I look to fully funding and deploying far more advanced systems, but I fully expect others to do so.

Do I miss the old days of PM cat-herding? $%^& no!

Is it worth it to get the project deployed? Definitely.
